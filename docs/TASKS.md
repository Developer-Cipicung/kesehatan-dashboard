- Mengikuti seluruh dokumentasi project.
- Reusable apabila memungkinkan.
- API telah terintegrasi (jika relevan).
- Dokumentasi telah diperbarui (jika diperlukan).
- Checklist pada TASKS.md telah diperbarui.
